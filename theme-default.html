HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: Task

  • id: required (integer)
  • title: required (string)
  • description: required (string)
  • frequency: required (integer)

    Number of days between two executions of this task

  • completionTime: required (integer)

    Time to be allowed for the task to be completed in hours

  • users: required (array of integer)

    All users that will have to complete this task at one point or another

  • currentTurn: required (union of Turn or nil)
  • upcomingTurns: required (array of Turn)

    Items: Turn

    • userId: required (integer)
    • startDate: required (datetime)

Example:

[
  {
    "id": 1,
    "title": "Doing the dishes",
    "description": "Use soap!",
    "frequency": 3,
    "completionTime": 5,
    "users": [
      1
    ],
    "currentTurn": {
      "userId": 1,
      "startDate": "2018-02-12T04:46:59.234975184Z"
    },
    "upcomingTurns": [
      {
        "userId" : 3,
        "startDate": "2018-02-15T07:00:23.233925184Z"
      }
    ]
  }
]
